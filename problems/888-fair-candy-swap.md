<p>爱丽丝和鲍勃有不同大小的糖果棒：<code>A[i]</code> 是爱丽丝拥有的第 <code>i</code>&nbsp;块糖的大小，<code>B[j]</code> 是鲍勃拥有的第 <code>j</code>&nbsp;块糖的大小。</p>

## Solution

Language: **python3**  /  Runtime: 112 ms

```python3
class Solution:
    def fairCandySwap(self, A, B):
        """
        :type A: List[int]
        :type B: List[int]
        :rtype: List[int]
        """
        n = (sum(A) - sum(B)) // 2
        A = list(map(lambda a: a -n, A))
        b = (list(set(A) & set(B)))[0]
        return [b + n, b]


```
