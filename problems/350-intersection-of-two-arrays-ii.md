## Solution

Language: **python**  /  Runtime: 112 ms

```python
class Solution(object):
    def intersect(self, nums1, nums2):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :rtype: List[int]
        """
        tmp = []
        for a in nums1:
            for b in nums2:
                if a == b:
                    nums2.remove(b)
                    tmp.append(a)
                    break
        return tmp
```
